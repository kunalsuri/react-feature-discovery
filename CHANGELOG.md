# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2025-01-11

### Added
- Initial release of React Feature Discovery
- Comprehensive React codebase analysis
- Support for TypeScript and JavaScript
- Multiple output formats (Markdown, JSON, HTML)
- Configurable categorization rules
- React pattern detection (hooks, components, contexts, HOCs)
- Dependency graph generation
- Full-stack application support
- Migration guidance generation
- CLI interface with multiple options
- Configuration file support (.rfdrc.json, .rfdrc.js, package.json)
- Technology detection for React ecosystem
- Performance optimizations (parallel processing, caching)

### Features
- Automatic file scanning and categorization
- Internal and external dependency analysis
- Complexity metrics calculation
- Migration notes generation
- Circular dependency detection
- Client/server code distinction
- Customizable environment patterns
- Flexible module type system
